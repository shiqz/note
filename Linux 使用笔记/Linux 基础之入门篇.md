### 一、Linux 的基本介绍
- 读法（里纽克斯、里那克斯）
- Linux 是一款操作系统，开源、免费、安全、高效、稳定、处理高并发强悍、部署企业级项目。
- 创始人（林纳斯）
- 吉祥物是一个企鹅
- 基于 Linux 内核的发行版系统（CentOS、Redhat、Ubuntu、Suse、红旗操作系统）

最开始贝尔实验室一起决定研发一款操作系统，但是效果不是很理想，这个系统最后以失败告终。`Ken tompson` 是 B 语言和 C 语言创始人（`Denins richres` 也是创始人之一），他也是一起参与研发的人之一，但是他没有放弃这个系统的研发，于是他又创造了 B 语言，然后通过 B 语言实现了这个操作系统，也就是 Unix。后来又用 C 语言重写了 Unix。

之后有很多其他商业公司在这个 Unix 内核的基础上开发了很多其他版本的操作系统，但是都是收费的，非常的贵。导致很多普通的技术爱好者无法使用，这个时候 `Richard Stallman` 发起了一个 GNU 计划，提倡软件自由的概念。我们基于这个开源协议就可以在上面发布和自由软件。

为了响应 GNU 计划，于是就创造出了 Linux 操作系统，第一版 Linux 是由 `Linus Torvalds` 发布的，而现在好用的 Linux 是被全世界的技术爱好者不断的完善和维护的。

Linux 可以说是 Unix 的发行版: `Unix => AT&T System V => Minix => Linux`；基于 Linux 内核的发行版系统：CentOS、Redhat、Ubuntu、Suse、Fedora、红旗操作系统。
