### 一、简介
Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。就本人而言也是非常喜欢它的，好多学习笔记都是用其进行撰写的，下面将简单介绍它的基本使用。

### 二、标题
#### 2.1 语法格式
```txt
#+空格+标题内容
```
#### 2.2 举例
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```
最多也就是6级标题，分别对应 HTML 中的 `h1 ~ h6`。
